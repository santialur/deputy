## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development.

### Prerequisites

You'll need to globally install the following packages.

* NodeJs
* Bower
* Sass
* Gulp

### Installing

```
$ npm install
```

```
$ bower install
```

Run the app:
```
$ gulp serve
```
